# LOEX 编码规范

### 代码规范
- 类顶部必须带有注释
  ```
  /**
   *  <类用途>
   * @author <作者>
   * @since <时间>
   */
   public class Test {}
   ```
- 代码空格使用
  * 类与实体之间只能使用一个空格
  * 参数之间只能使用一个空格
  * 括号内无需使用空格
  ```
    Test test = new Test();
    test.print("a", "b");
  ```
- 提交代码前必须使用```Ctrl+Shift+F```进行代码格式化

- SQL脚本规范
  * ${系统名称}\_${版本号}\_${日期}.sql
  * 脚本内容不能包含drop、delete、truncate命令
  * 所有脚本必须是增量脚本
  * 所有脚本必须包含业务注释，注明具体业务为什么需要执行此脚本
